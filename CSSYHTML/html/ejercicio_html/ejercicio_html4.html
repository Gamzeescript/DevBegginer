Pasos para aprender

  • Elegir el tema a aprender
  • Investigar como me siento mejor al aprender:
    • De forma virtual
    • De forma auditiva
    • De forma kinestésica
  • Repasar lecciones:
    • Repetirlo conmigo mismo en pocas palabras
    • Decirle a otros y saber si queda claro
    • Haciendo preguntas y pruebas:
      • Se vale equivocarse, pero se debe conocer por qué
      • Prueba ser curioso y saber si existe solo una forma o una manera de ser
  • Anota o escribe, almacena la nueva información
  • Repite todo el proceso para nuevas lecciones
